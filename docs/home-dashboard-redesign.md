# 首页重构说明

## 目标

修复首页 `null`、空接口状态和旧卡片样式问题，改为更现代的概览页，并在首页提供软件源快捷切换。

## 已实施

1. 首页布局改为：
   - 顶部 Hero 概览
   - 软件源快捷切换面板
   - `CPU / 内存 / 路由条目` 圆形指标环
   - 实时流量折线图
   - 网络概览与接口状态
2. 去掉对缺失 RPC `luci.getETHInfo/getTempInfo` 的依赖。
3. 接口状态改为基于 `network.interface.status` 展示 `WAN/WAN6/LAN/WWAN`。
4. 软件源支持首页一键切换：
   - `中科大源`
   - `官方源`

## 设计取舍

1. 当前优先保证界面稳定和数据可用，不再依赖仓库中不存在的后端 RPC。
2. “网口状态”语义调整为“接口状态”，避免在缺少底层交换机 RPC 时整块无数据。
3. 软件源快捷切换直接改写 `/etc/opkg/distfeeds.conf`，保证即时生效。
